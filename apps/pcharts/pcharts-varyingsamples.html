p-Chart Trainer: Varying Sample Sizes

This version uses different sample sizes \(n_i\) for each subgroup and variable control limits UCL\(_i\), LCL\(_i\). Students see why p‑charts are especially useful when sample size changes.

Step 1 · Scenario setup

Each day you inspect a different number of units and count how many are defective. For subgroup \(i\): \(p_i = d_i / n_i\), and the overall \(\bar p\) uses total defectives divided by total inspected.

Formulas with variable \(n_i\)

Overall fraction defective:

\(\bar p = \dfrac{\sum d_i}{\sum n_i}\)

For subgroup \(i\):

\(p_i = \dfrac{d_i}{n_i}\)

\(\text{UCL}_i = \bar p + 3\sqrt{\dfrac{\bar p(1-\bar p)}{n_i}}\)

\(\text{LCL}_i = \bar p - 3\sqrt{\dfrac{\bar p(1-\bar p)}{n_i}}\)

If \(\text{LCL}_i < 0\), set \(\text{LCL}_i = 0\).
